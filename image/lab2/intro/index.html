Lab 2 - Introduction

Objectives

Running the Raspberry Pi OS on a virtual machine on host

  1. Getting the Raspberry Pi OS (formerly Raspbian) image
  2. Installing Virtual Machine (QEMU VM) software on the host
  3. Preparing the image to run it in the QEMU VM
  4. Executing QEMU VM
  5. Installing ROS2 from source code or using docker

Note

The host or the host computer is the physical machine on which you will execute the Raspberry Pi OS. In this lab it is your laptop.
